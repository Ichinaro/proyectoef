Los modelos son clases que contienen la estructura de una tabla en la DB
En este punto creamos 2 modelos: Categoria.cs y Tarea.cs


### Función de `Guid`:

  **Identificador Único Global (GUID)**:
   - `Guid` es una estructura en .NET que representa un identificador único global.
   - Se utiliza para identificar de manera única objetos o registros en una base de datos.


### Función de `virtual`

  - `virtual` comunmente se usa para implementar carga diferida ( Lazy loadign )
  - esto permite que los datos se cargen solo cuando se requieran ( o se acceda a esta propiedad por primera vez )
